# Task-5-EDA
Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python, Pandas, Matplotlib, and Seaborn. The project includes statistical analysis, visualizations, and a summary report.
## Objective
The objective of this task is to perform Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python. The analysis helps identify patterns, trends, correlations, and outliers through statistical summaries and visualizations.
## Tools Used
- Visual Studio Code (VS Code)
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- Sample Superstore Dataset

## Project Files
- EDA_Superstore.ipynb – Jupyter Notebook containing the EDA code.
- Sample - Superstore.csv – Dataset used for analysis.
- EDA_Report.pdf – PDF report with observations and findings.
- README.md – Project documentation.

## Exploratory Data Analysis Performed
- Loaded the dataset
- Displayed dataset information using `info()`
- Generated statistical summary using `describe()`
- Checked for missing values
- Counted category-wise records using `value_counts()`
- Created the following visualizations:
  - Histogram
  - Box Plot
  - Scatter Plot
  - Correlation Heatmap
  - Pair Plot

## Key Findings
- Most sales are of smaller amounts, while a few transactions have very high sales.
- Profit contains several outliers.
- Sales and Profit show a moderate positive relationship.
- Higher sales do not always result in higher profit.
- Discount negatively impacts Profit.

## Outcome
Successfully performed Exploratory Data Analysis (EDA) using Python and visualized important patterns, trends, and relationships in the Superstore dataset.
